Optimización Turbo Extrema
Mata 30+ procesos innecesarios, pausa servicios de Windows y configura el registro para máxima prioridad al juego.

Extensión de Brave
Inyecta configuración de FPS y pixel directamente en Bloxd.io via content script. Sin modificar el juego.

Detecta RAM, núcleos de CPU y modelo de GPU. Aplica flags de Brave y tweaks distintos para cada nivel de hardware.
